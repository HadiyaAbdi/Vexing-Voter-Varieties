# Voter Turnout Prediction using Logistic Regression

This project investigates factors associated with voter participation using logistic regression and survey data from **FiveThirtyEight** and **Ipsos**. The goal is to identify demographic and political characteristics associated with infrequent voting while evaluating the predictive performance of a classification model.

## Project Overview

The analysis estimates the probability that an individual is an infrequent voter ("rarely/never") based on demographic, socioeconomic, and political variables. Model performance is assessed using logistic regression, odds ratios, confidence intervals, and ROC analysis.

## Research Questions

- How does political party affiliation influence the likelihood of being an infrequent voter?
- Which demographic and socioeconomic characteristics are associated with lower voter participation?
- Can logistic regression accurately predict likely infrequent voters?

## Data

This project uses publicly available survey data collected by **FiveThirtyEight** in collaboration with **Ipsos**. The analysis is fully reproducible, with datasets downloaded directly within the R code rather than stored in this repository.

**Variables include:**
- Age
- Education
- Race
- Gender
- Household Income
- Political Party Affiliation
- Voting Frequency
